**GenZInvest** is a hyper-personalized, gamified financial literacy platform designed for the next generation of Indian investors. We strip away the "boomer jargon" and replace it with a social-first experience that makes building wealth as addictive as scrolling through Instagram.

---

## 🌟 Key Features

### 1. 📊 Trader Social Feed (Community)
- **Live Social Timeline**: Share chart setups, investment ideas, and market insights.
- **Engagement Loop**: Double-tap to like, comment on colleague's trades, and earn XP for every interaction.
- **In-Memory Prototype Engine**: Lightning-fast performance using an optimized server-side in-memory storage system.

### 2. 🔥 Gamification & Streaks
- **Daily Habits**: Maintain your 100-day streak to unlock exclusive rewards.
- **XP Ecosystem**: Earn XP for learning, posting, and checking news.
- **Partner Rewards**: Redeem your XP for real cash in partnered broker wallets like **Upstox** and **INDmoney**.

### 3. 🧠 Smart Planning Tools
- **SIP Planner**: A multi-step onboarding wizard that maps your income & goals to a beginner-friendly mutual fund allocation.
- **Portfolio Simulator**: Trade in a risk-free environment with ₹100,000 of virtual "Monopoly Money" using real-time mock price data.
- **What-If Calculators**: Visualize the power of compound interest vs. the risk of panic selling during market crashes.

### 4. 🤖 AI Financial Assistants
- **Personalized News**: Our AI analyzes every headline and tells you *exactly* why it matters to your specific risk profile.
- **AI Tax Advisor**: Talk to "Tax Scholar," your digital CA, to navigate the complexities of Section 80C and the Indian tax regimes.
- **AI Term Explainer**: One-click GenZ-friendly definitions for any finance term (NAV, Expense Ratio, etc.).

## 🛠️ Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router & Server Actions)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) & [Framer Motion](https://www.framer.com/motion/) for premium animations.
- **Authentication**: [Firebase Auth](https://firebase.google.com/docs/auth) (Google One-Tap Login).
- **Core AI**: [Groq API](https://groq.com/) (Llama 3 8B & 70B) for sub-second inferencing.
- **State Management**: React Context API (Gamification & Theme).

## 🏃‍♂️ Getting Started

### Prerequisites
- Node.js 18.x or later
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Suraj280807/genzinvest01.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables in `.env.local`:
   ```env
   GROQ_API_KEY=your_groq_api_key
   # Firebase variables are already linked in lib/firebase.ts
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
